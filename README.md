# **Predictive Maintenance System**

A machine learning–based predictive maintenance system that analyzes sensor data to detect potential equipment failures. It predicts components at risk, classifies failure probabilities, and provides maintenance recommendations through an interactive dashboard.

---

## **Tech Stack**

**Machine Learning:** Python, Scikit-learn, Pandas, NumPy
**Backend:** Flask
**Frontend:** HTML, CSS, JavaScript
**Visualization:** Matplotlib / Plotly
**Data:** Sensor dataset (historical + real-time simulation)

---

## **Features**

* Failure prediction using ML models
* Component risk classification
* Sensor data analysis & preprocessing
* Interactive dashboard for insights
* Maintenance recommendations

---

## **Project Structure**

* `app.py` – Flask backend
* `models/` – ML model files
* `static/` – CSS, JS
* `templates/` – HTML pages
* `data/` – Sensor dataset
